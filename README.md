# Monster Slayer 🗡️

Un jeu de combat au tour par tour développé en HTML, SASS et JavaScript.

## 🛠️ Technologies

- HTML5
- SASS/CSS
- JavaScript ES6+

## 🚀 Lancement

1. Ouvrir `index.html` dans votre navigateur
2. Pour compiler le SASS :
   ```bash
   sass stylesheets/main.scss main.css --watch
   ```

## 📁 Structure du projet

